# Comparador de Preços de Supermercado

Aplicação web responsiva para comparação de preços de produtos de supermercado,
permitindo ao usuário montar sua lista de compras e encontrar o menor custo possível,
inclusive combinando diferentes mercados.

## Stack
- React + Vite
- Node.js + Express
- SQLite
- JWT

## Como rodar

### Backend
cd backend  
npm install  
npm run dev  

### Frontend
cd frontend  
npm install  
npm run dev  

## Funcionalidades
- Login com aprovação administrativa
- Cadastro de produtos e mercados
- Comparação de preços
- Lista de compras
- Exportação em PDF
- Sistema de pontos
